# 音频文件说明

将以下音频文件放在此目录中启用游戏音效：

## 必需的音频文件：

1. **eat.mp3** - 吞噬食物音效
   - 建议时长：0.5秒
   - 音效类型：清脆的"波"声

2. **skill.mp3** - 技能释放音效
   - 建议时长：1秒
   - 音效类型：能量释放声

3. **death.mp3** - 死亡音效
   - 建议时长：1.5秒
   - 音效类型：爆炸/消失声

4. **bgm.mp3** - 背景音乐
   - 建议时长：循环
   - 音乐类型：太空氛围音乐

## 音频资源推荐：

- 免费音效网站：
  - Freesound.org
  - Zapsplat.com
  - Mixkit.co

- 免费音乐网站：
  - Pixabay Music
  - Bensound
  - Incompetech

## 注意：

- 音频文件会自动加载，如果文件不存在，游戏仍可正常运行（无声音）
- 建议使用MP3格式以获得最佳兼容性
- 音量已在代码中调整，无需额外处理
